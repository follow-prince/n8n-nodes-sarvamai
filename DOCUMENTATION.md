# Sarvam AI Credentials Documentation

This document explains how to set up credentials for the Sarvam AI node in n8n.

## Prerequisites

To use this node, you need a Sarvam AI API subscription key.

## Creating an API Key

1. Go to the [Sarvam AI Dashboard](https://dashboard.sarvam.ai/).
2. Sign up or log in to your account.
3. Navigate to the API keys section.
4. Create a new API key or use an existing one.

## Using the API Key in n8n

1. In your n8n instance, create a new Sarvam AI credential.
2. Enter your API Key in the "API Key" field.
3. Save the credential.

For more information, visit the [official Sarvam AI documentation](https://docs.sarvam.ai/).
